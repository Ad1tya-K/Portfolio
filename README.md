#My Portfolio Website

A futuristic terminal-style portfolio website built using HTML, CSS, and JavaScript with Matrix-inspired animations, typing effects, responsive sections, and an interactive certifications carousel. The website showcases personal information, skills, projects, experience, certifications, and contact details in a visually engaging cyberpunk-inspired interface.
